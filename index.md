##如何练习本站的流程
本站的主要面向对象是有一定基础的人族玩家(推荐钻石组及以上)，基础比较薄弱的玩家可以考虑选一两个稳健型和经济型的战术好好练习，快速提高基本功，而对于较强的玩家，也建议多掌握几个流程。
为了叙述简洁，本站的流程并不会像一些传统的流程一样，事无巨细地给出多少人口造什么，怎么造之类的，而是大多只给出最关键的建筑/产兵/科技等部分。记忆几十个人口节点是非常累的，而且缺乏弹性(基础弱的玩家很难做到细节一一抠死)。所以我选择用一种类似流水帐叙事的方法表述一个简化后的流程。
所以要自己练习的话，首先要对照着录像大概过一遍流程，看下职业选手是怎么执行流程的，记下一些流程检查点，比如[6:00有100人口]、[8:00有至少6架运输机+160人口]这样。
接下来就是模仿练习了，这个过程很长，主要目的是培养肌肉记忆。练习方式无非分两种，对着电脑练习和和真人练习。
1. 对着电脑练习，一般建议开经济型电脑，稍微像真人一些。基础弱的玩家可以把电脑难度调低，强的玩家可以开作弊三或者两家以上电脑来自我挑战(主要是练习双线简单骚扰+运营)。对着电脑练习主要是为了减少真人玩家会对练习造成的干扰。过程中要仔细对照流程重点和自己记下的检查点，优化流程细节，最终做到可以不假思索地完成整套流程，并无太多瑕疵。
1. 和真人练习，并无所谓自定义、非排名还是排名天梯。这种练习更注重于意识和理解的培养：意识一般是体现在侦查、骚扰和推进时机上；而理解则在于体会流程背后的原理(如为什么有时先三四气再VS，为什么有时不出解放出运输机)，逐渐做到融汇贯通。通过和真人练习，能更好地把别人的流程转化成自己的，还可以加入一些属于自己的独特改动。

当流程逐渐形成肌肉记忆之后，就可以上天梯实战了。

##常规开局速览
---
接下来列出的开局是在比赛中非常常见的几个开局。
###裸双
在进入虚空之遗时代后，裸双开局就非常非常少见了，只在某些相当追求经济的流程中才会出现裸双开。
由于成体系的流程过少，目前没有一个通用的裸双模板。
``` {.line-numbers}
基地生产的第一个农民建房子
#16农民集结点点在二矿 等400矿双开
```
>由于裸双实在过于少见，一般只出现在专门设计的流程里(多为机械化)。
###速气1-1-1开局
现在非常常见的TvT标准开局，流程在虚空初期便已经成型，但是直到渡鸦开局出现后，才逐渐代替速VS空投/女妖开成为主流开局。
``` {.line-numbers}
基地生产的第一个农民建房子
有75矿下#1气
有150矿下BB
有75矿下#2气
#16农民探路
BB好后 死神 下VF 基地变形星轨 同时双气改为3农民采集(1+2)，拉其中一个农民去建房子
那个建房子的农民建好后回到只有1农民采集的气矿上
主矿16农民满采矿后把集结点分别设在两个气上 使之满采
出一死神一火车 有钱开矿
29人口时建一个房子
有150矿下VS
视个人习惯选择出(死神、机枪)与(火车、地雷)
```
>和较早时候常见的速VS的流程相比，这一版开局在经济和科技之间找到了一个不错的平衡。
###开矿型1-1-1开局
开矿型1-1-1开局，即兵营-二基地-重工-机场开局。
笔者一般习惯使用**慢气兵营开矿**和**先气兵营开矿**表示开矿型1-1-1开局的两个常见变种。

####慢气兵营开矿
``` {.line-numbers}
基地生产的第一个农民建房子
房子建好后下一个BB 拉一个农民去气矿旁边等75矿下气矿
BB好后生产死神 主基地变形星轨 造BB的农民去二矿处等400矿下#2基地
大概有50矿的时候拉一个主矿农民建房子
大概有90气的时候拉一个主矿农民在BB旁边下VF
死神好后出一个机枪
有75矿的时候下#2气矿 机枪好后BB挂双倍
二基地变形星轨
VF好后起飞 造VF的农民下VS 
BB的双倍好后起飞 挂件给VF BB落下建造科技挂件
```
####先气兵营开矿
``` {.line-numbers}
基地生产的第一个农民建房子
大概有50矿的时候拉一个农民造下气矿
房子建好后农民等150矿下一个BB
BB好后生产死神 主基地变形星轨 造BB的农民去二矿处等400矿下#2基地
有大概100气时拉一个农民下VF
死神好后BB挂双倍
主基地星轨好后造的第一个农民下一个房子
有75矿的时候下#2气矿
二基地变形星轨
VF好后起飞 造VF的农民下VS
BB的双倍好后起飞 挂件给VF BB落下建造科技挂件
```
> 慢气开的优势是二基地能早3~5秒钟建下，并且能出一个机枪防TvZ时偶尔会遇到的绕两条狗打法，还能稍微封下对面的侦察，属于比较稳健的打法；而先气开则能早一些双产VF部队，常用于TvP的投雷开，与TvZ的速投4火车开。

###标准速三开
``` {.line-numbers}
基地生产的第一个农民建房子
房子建好后下一个BB 拉一个农民去气矿旁边等75矿下气矿
BB好后生产死神 主基地变形星轨 造BB的农民去二矿处等400矿下#2基地
大概有50矿的时候拉一个主矿农民建房子
大概有90气的时候拉一个主矿农民在BB旁边下VF
死神好后出一个机枪
机枪好后BB挂双倍
在2:40时 一般能有400矿 拉一个主矿农民下#3基地
```
>标准速三开现在只多见于TvZ中。后续变化较多(典型的有Bomber流、卫星流、TY流)。缺点是很怕两矿Rush，优点是操作量不大，只要运营足够扎实就可以一拳打死。
